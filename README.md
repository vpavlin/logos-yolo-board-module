# logos-yolo-board-module

> **Moved.** This module is now `module/` inside the [`vpavlin/logos-yolo`](https://github.com/vpavlin/logos-yolo) monorepo.
>
> This repository is kept read-only for history. All new work — issues, PRs, commits — happens in the monorepo.

The backend module (`yolo_board_module`) owns Yolo's polling loop, message/attachment cache, subscription persistence, storage-upload orchestration, and per-message Waku threads. The QML UI plugin it pairs with (formerly `zone-sdk-test`) lives at `ui/` in the same monorepo.

## Quickstart (new location)

```bash
git clone https://github.com/vpavlin/logos-yolo
cd logos-yolo
# follow the README there
```
